\doxysubsection{Detailed Description}
Utility for encrypting, decrypting, and breaking Caesar ciphers using frequency analysis. 


\begin{DoxyItemize}
\item This program provides tools to analyze text, calculate letter distributions, and perform automated decryption using various distance metrics. 
\end{DoxyItemize}
